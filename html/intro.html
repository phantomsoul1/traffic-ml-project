Project Introduction & Documentation

Information on our Project's Dataset and Machine Learning Models Used

The Project

This project was developed for Rutger's University Data Science Bootcamp students in Group 6. The members of Group 6 include Jake Byford, George Hyros, Robert Peron, and Michael Seifer.


The goal of our project was to understand what weather factors and conditions have the largest influence on the severity of accidents across the United States. We used a dataset on Kaggle that had approximately 3.5 million records of U.S. Traffic Accidents from February 2016 until June 2020. The accident data itself comes from many different entities, including US and State Departments of Transportation, Traffic Cameras and Traffic sensors.


Using the dataset provided on Kaggle, we were able to create and deploy two separate Machine Learning models. In order to be able to deploy the models and present them, we used the following tools to perform back-end and front-end operations.

  • Amazon AWS (RDS & S3): Used Amazon Relational Database Service to set up a PostgreSQL database in the cloud and S3 service in order to store the U.S. Traffic Accident csv file.

  • SQL: To create the table schema to import our data into a table in the database.

  • Python/Spark: For cleaning our data we used Pandas library in Python. For importing the data to the database table, we used Spark to connect to the S3 object where we preprocessed the data to import to our database. We also used the Scikit and Numpy libraries in Python in order to deploy our classification Machine Learning models and record the results.

  • Tableau: In order to dynamically visualize our dataset and Machine Learning Model results, we have embedded visualizations from Tableau in our webpage.

Data Processing & Analysis

Understanding the relationship that certain driving and weather conditions have with the severity of a traffic accident has serious importance to not only the safety of U.S. drivers, but for first responders and dispatchers. Emergency Services teams can understand what given weather and driving conditions are present in order to understand when additional teams may be required and what medical equipment may be necessary for more severe accidents.


In order to prep the data for use in our Machine Learning models, the following steps were done for the data cleanup:

  • Loading the csv file in a Pandas Dataframe
  • Renaming and dropping columns from dataset
  • Converting datatypes in various columns
  • Creating random samples from the dataset for ML models
  • Removing NAN values from dataset for ML Models

Machine Learning Method

For our project goal, since we decided to analyze and better understand what factors into accident severity, our team knew we would be limited in the Machine Learning models available to us to use. Since we have the severity of each accident in our dataset, our data has a determined outcome and therefore any Machine Learning model we run would have use a Supervised Machine Learning model.
